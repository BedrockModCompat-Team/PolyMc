PolyMc

The power of a modded server, without a modded client

Item Polys

At a basic level item polys take in the servers sided item and output one to be displayed on the client. The default item poly, the CustomModelDataPoly, works quite well in most cases. All of PolyMc's item polys are based on the CustomModelDataPoly. The biggest reason for making a custom item poly is to customize the resources that get stored for that poly.

Example: changing the item used on the client

public void registerPolys(PolyRegistry registry) {
    registry.registerItemPoly(moddedItem, new CustomModelDataPoly(registery.getCMDManager(), moddedItem, Items.DIAMOND_SWORD))
}

note: if your modded item is damageable, replace CustomModelDataPoly with DamageableItemPoly
This will Poly your item as usual, just using a diamond sword instead of a stick. It will automatically allocate a CMD value for it.
